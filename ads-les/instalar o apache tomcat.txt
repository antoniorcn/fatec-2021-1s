Instalar o Apache Tomcat

1) Baixar o apache tomcat https://downloads.apache.org/tomcat/tomcat-9/v9.0.43/bin/apache-tomcat-9.0.43.zip

2) Descompacte o arquivo

3) Abra um prompt de comandos

4) Acesse o diretorio apache-tomcat-9.0.43\bin

5) Execute o startup.bat

// Se ocorrer erros
